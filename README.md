# M68000 Assembler

* type make to build m68.

## Usage: 
```bash
	./m68 [INFILE] [OUTFILE]
```

## Utils
* hwrite - Program for writing bytes by hand to ouput file
### Usage
```bash
	cd utils
	make
	./hwrite > [BINFILE]
	make testbin bin=[BINFILE]
```
